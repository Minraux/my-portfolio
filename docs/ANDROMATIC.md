# Andromatic — 10-Step Sequencer

Веб-реализация аналогового секвенсора Andromatic (Эркки Куренниеми, 1968).

**URL:** https://dobrovolski.space/andromatic

## Описание

Andromatic — это 10-шаговый паттерн-секвенсор с аналоговым звучанием. Веб-версия воссоздаёт дизайн и функциональность оригинального устройства 1968 года.

## Функционал

### Секвенсор
- **10 шагов** — каждый шаг может быть активен/неактивен
- **5 октав** — 50 нот (A2–C7)
- **Регулировка высоты** — кнопки ▲/▼ для каждой ноты
- **Визуализация** — индикатор текущего шага (оранжевая точка)

### Управление звуком
| Контроллер | Диапазон | Описание |
|---|---|---|
| **TEMPO** | 10–220 BPM | Скорость секвенсора (восьмые ноты) |
| **ATTACK** | 0.001–3 сек | Атака огибающей |
| **DECAY** | 0.05–10 сек | Затухание огибающей |
| **FILTER** | 100–8000 Гц | Частота низкочастотного фильтра |
| **VOLUME** | 0–1 | Общая громкость |

### Волновые формы
- **Sine** (синусоида)
- **Triangle** (треугольная)
- **Sawtooth** (пилообразная)
- **Square** (меандр)

### Shuffle
Рандомизация активного паттерна. При включении каждый шаг получает случайное состояние (50% вероятность).

## Аудиодвижок

### Цепочка сигнала
```
Осциллятор → Filter (LPF) → Gain (ADSR) → Master Gain → Waveshaper → Compressor → Limiter → Analyser → Выход
```

### Обработка
- **Waveshaper** — мягкое насыщение (tanh, 10% wet)
- **Compressor** — динамическая компрессия (threshold: -18dB, ratio: 8:1)
- **Limiter** — ограничение (tanh waveshaper, 4x oversample)

### Расчёт темпа
```
Интервал шага (мс) = 30000 / BPM
```
Формула основана на восьмых нотах: при 120 BPM интервал = 250мс (120 восьмых нот в минуту).

## Технические детали

### Компоненты
- `AndromaticClient.tsx` — основной клиентский компонент
- `page.tsx` — серверная обёртка с метаданными

### Стили
- Страница изолирована от глобальных стилей через класс `.andromatic-page`
- Хедер сайта скрыт на странице `/andromatic`

### Аудио API
- Web Audio API
- AudioContext (авто-старт при первом взаимодействии)
- OscillatorNode (4 волны)
- BiquadFilterNode (lowpass)
- GainNode (ADSR + master volume)
- WaveShaperNode (saturation)
- DynamicsCompressorNode
- AnalyserNode (осциллограф)
