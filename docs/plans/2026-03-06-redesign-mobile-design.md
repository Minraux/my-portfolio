# Редизайн: мобильная версия и фото в герое

**Дата:** 2026-03-06
**Статус:** утверждён

## Проблемы (исходные)

1. Нет мобильной версии — сайт не адаптирован под маленькие экраны
2. Хедер ломается на мобиле — имя переносится, навигация уезжает за экран
3. Нет бургер-меню для мобайл/планшет
4. H1 в статьях слишком крупный на мобиле
5. Соцсети уезжают на второй экран — не видны без скролла
6. Нет возможности добавить фото на главную

## Цветовая схема

| Переменная         | Значение                  | Использование                          |
|--------------------|---------------------------|----------------------------------------|
| `--color-bg`       | `oklch(14% 0.008 60)`     | Фон всего сайта (~#1C1C1E, тёмно-серый с тёплым тоном) |
| `--color-hero-fallback` | `oklch(96% 0.02 85)` | Фон героя когда нет фото (кремово-яичный) |
| `--color-accent`   | `oklch(72% 0.12 55)`      | Акцентный: тёплый золотистый           |
| `--color-text`     | `oklch(95% 0 0)`          | Основной текст                         |
| `--color-text-muted` | `oklch(60% 0 0)`        | Второстепенный текст                   |

## Герой (главная страница)

### С фото
- Фото занимает весь первый экран (`100svh`), `object-fit: cover`
- Поверх — тёплый полупрозрачный оверлей (`linear-gradient(to top, rgba(0,0,0,0.75) 0%, rgba(0,0,0,0.2) 60%, transparent 100%)`)
- Текст и соцсети — в нижней части, поверх оверлея, белый текст
- Мобайл: фото масштабируется под экран (cover)

### Без фото
- Фон: кремово-яичный `--color-hero-fallback`
- Текст тёмный (charcoal)
- Та же компоновка — имя, подпись, соцсети внизу

### Компоновка героя
```
Mobile / Desktop:
┌──────────────────────────────┐
│  [ФОТО — cover / fallback]  │
│                              │
│  (оверлей снизу вверх)      │
│                              │
│  ВЛАД ДОБРОВОЛЬСКИЙ          │  ← Onest, крупно
│  Звуковой художник           │  ← Cormorant italic
│  [tg] [ig] [bc]              │  ← SVG иконки 20px
└──────────────────────────────┘
```

## Хедер

| Брейкпоинт     | Поведение                                                       |
|----------------|-----------------------------------------------------------------|
| < 768px        | Имя слева, иконка бургера справа. По клику — fullscreen overlay с навигацией |
| ≥ 768px        | Текущий вид: имя слева, ссылки справа                          |

- `mix-blend-difference` остаётся
- Burger overlay: тёмный фон, пункты меню крупно по центру, кнопка закрытия

## Соцсети — иконки

Три платформы: **Telegram, Instagram, YouTube**
Реализация: inline SVG-компонент `SocialIcon`, иконки нарисованы вручную (минималистично, 20×20px viewBox)

## Изменения в Sanity-схеме

### `settings.ts`
Добавить поле:
```ts
defineField({
  name: 'heroImage',
  title: 'Фото на главной',
  type: 'image',
  options: { hotspot: true },
})
```

### `settings.ts` — socials
Добавить поле `icon` в объект соцсети:
```ts
{ name: 'icon', type: 'string', title: 'Иконка', options: {
  list: ['telegram', 'instagram', 'youtube']
}}
```

## Форма обратной связи

Страница `/contact` (или секция на `/about`).

**Стек**: Resend — бесплатно до 3000 писем/мес, простая интеграция с Next.js.

**Поля формы**: Имя, Email, Сообщение, кнопка «Отправить»

**Реализация**:
- `app/api/contact/route.ts` — POST-handler, вызывает Resend API
- Письмо приходит на `laskalugas@gmail.com`
- ENV переменная: `RESEND_API_KEY`
- Валидация на клиенте + серверная проверка
- Состояния: loading / success / error

**Размещение**: отдельная страница `/contact`, ссылка добавляется в навигацию хедера.

| Файл                              | Изменение                           |
|-----------------------------------|-------------------------------------|
| `app/(site)/contact/page.tsx`     | Новая страница с формой             |
| `app/api/contact/route.ts`        | API route для отправки через Resend |
| `components/Header.tsx`           | Добавить пункт «Контакт» в nav      |

## Что не меняется

- Общий стиль (тёмная тема, белый текст)
- Шрифты (Onest + Cormorant)
- Анимации FadeIn и PageTransition
- Страницы works, about, teaching, blog (кроме типографики H1)
