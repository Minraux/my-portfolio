# АНС Синтезатор — Горячие клавиши

## Транспорт и воспроизведение

| Клавиша | Действие |
|---------|----------|
| **Space** | Play / Stop |
| **⊙** (кнопка) | Follow mode (вкл/выкл автоскролл) |

## Инструменты и работа с canvas

| Клавиша | Действие |
|---------|----------|
| **D** | Рисовать (Синус/Треугольник/Пила по текущему выбору) |
| **E** | Стирать |
| **[** | Кисть −1 px |
| **]** | Кисть +1 px |
| **F** | Fit Zoom (вписать в окно) |

## Слайдеры и управление

| Элемент | Диапазон | Начальное значение |
|---------|----------|-------------------|
| Масштаб (Zoom) | 5–500% | 100% |
| Длительность | 1–120 мин | 30 мин |
| Громкость | 0–100% | 70% |
| Кисть | 1–40 px | 4 px |

## Кнопки и действия

| Кнопка | Действие | Группа |
|--------|----------|--------|
| **▶ Пуск** / **◼ Стоп** | Play/Stop | Транспорт |
| **⊙ Следить** | Toggle автоскролл | Транспорт |
| **Рисовать** | Режим рисования | Инструменты |
| **Стирать** | Режим стирания | Инструменты |
| **Очистить** | Очистить canvas | Инструменты |
| **Вписать** | Fit zoom | Масштаб |
| **Синус** | Sine wave | Тембр |
| **Треуг** | Triangle wave | Тембр |
| **Пила** | Sawtooth wave | Тембр |
| **Сохранить** | Download PNG | Файл |
| **Импорт** | Load PNG | Файл |

## Мышь и взаимодействие

| Действие | Результат |
|----------|-----------|
| **Hover над canvas** | Виден preview кисти + горизонтальная направляющая + нота (Hz) |
| **Drag по canvas** | Рисование линии |
| **Клик на линейке** | Перемотка плейхеда (scrub) |
| **Drag по линейке** | Непрерывная перемотка |
| **Mouse leave** | Очистка preview и направляющей |

## Дополнительно

### Hz-ридаут
Показывает текущую частоту при hover:
```
A4 · 440 Гц
C5 · 523 Гц
F#2 · 92 Гц
```

### Status в хедере
Во время воспроизведения:
```
0:32 / 30:00   (текущая позиция / общая длительность)
```

### Индикатор "Следить"
- **Включен** (⊙ с тёмным фоном) → плейхед тянет вид
- **Выключен** (⊙ обычный) → свободный скролл

### Автосохранение
- Сохраняется в localStorage каждые 2 сек после рисования
- При открытии: появляется бар восстановления (если есть сохранённая версия)

## Примеры использования

### Быстрый старт
1. Нарисовать несколько линий
2. Нажать **Space**
3. Слушать результат

### Изменение тембра
1. Нарисовать фигуру
2. Нажать **Пила** (или другой тембр)
3. Нажать **Space** → услышать другой звук

### Сохранение и восстановление
1. Нарисовать партитуру
2. Нажать **Сохранить** → скачивается PNG
3. Позже загрузить PNG → автоматически восстановится скорость
4. Закрыть вкладку → открыть → нажать **Восстановить** для автосохранённой версии

### Прецизионная работа
- **F** → fit zoom для общего вида
- Zoom слайдер → увеличить нужную область
- **Hover** → нота помогает выбрать правильную высоту
- **[/]** → быстро менять размер кисти
