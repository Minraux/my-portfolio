## 2. Правила hover (глобальные, только CSS)

Единое правило для всех интерактивных элементов — **инверсия цвета, transition: all 0.1s**.  
Никакого Framer Motion для hover.

| Элемент | Default | Hover |
|---|---|---|
| Pill-кнопка / nav-link | border white, bg transparent, text white | bg white, text black |
| Back-кнопка ‹ (круг) | border white, bg transparent, text white | bg white, text black |
| Работа на canvas (pill) | border white, bg black, text white | bg white, text black + preview сверху |
| Строка списка (мобайл/публикации) | bg transparent, title white, стрелка #444 | bg #0f0f0f, title #00FF41, стрелка #00FF41 |
| Текстовая ссылка | text white, underline #444 | text #00FF41, underline #00FF41 |
| CTA "Контакт" | всегда bg #00FF41, text black | без изменений |
| Активный пункт меню | bg #00FF41, text black | без изменений |

## 3. Хедер (components/Header.tsx)

### Десктоп (≥ 768px)
- Горизонтальная строка pill-кнопок, `padding: 12px 16px`, `gap: 8px`
- Пункты: **Работы · Публикации · Источник · Автор** — pills с `border: 1.5px solid white`, `border-radius: 9999px`, `padding: 6px 16px`, `font-size: 15px`
- Активный пункт — залит `--color-accent`, text black
- CTA **Контакт** — всегда залит `--color-accent`, прижат вправо (`margin-left: auto`)
- **Back-кнопка** — отдельный кружок 32×32px (`border-radius: 50%`), появляется только когда `pathname !== '/'`, стоит левее всех пунктов

### Мобайл (< 768px)
- Та же строка pills, горизонтальный скролл (`overflow-x: auto`, скроллбар скрыт)
- Бургер-меню **не нужно**

### Прозрачность на главной
```js
// Если pathname === '/' и scrollY < 50:
header background: transparent
// Иначе:
header background: rgba(0,0,0,0.95), backdrop-filter: blur(8px)
```

### Убрать
- `mix-blend-difference`
- Все Framer Motion анимации хедера
- Имя "ВЛАД ДОБРОВОЛЬСКИЙ" из хедера (убрать полностью — в хедере только навигация)

## 5. Страница Работы — freeform canvas (app/(site)/works/page.tsx)

### Десктоп
- Контейнер: `position: relative`, `min-height: 200vh`, `width: 100%`, `background: #000`
- Каждая работа — pill-лейбл (`border: 1.5px solid white`, `border-radius: 9999px`, `padding: 8px 20px`, `font-size: 15px`)
- **Без emoji** — только текст

### Алгоритм размещения (без конфликтов)
Работы сортируются по `year desc` (новые первыми).  
Canvas делится на горизонтальные зоны по годам: новые — верхняя треть, старые — нижняя.  
При добавлении каждого лейбла — проверка bbox (`top/left/width/height`) на пересечение с уже размещёнными.  
Если пересечение — сдвиг вправо или вниз в пределах своей зоны.

```ts
// Структура позиции
type WorkPosition = { top: string; left: string } // в процентах

// Предвычисленные позиции хранятся в Sanity (поля canvasTop, canvasLeft)
// или генерируются на клиенте при первом рендере и кешируются в localStorage
```

### Hover на canvas
- При наведении: pill инвертируется (bg white, text black)
- Появляется preview-блок: `position: absolute`, `bottom: 110%`, `left: 0`, `width: 200px`, `height: 150px`, `opacity: 0 → 1`, `transition: opacity 0.15s`, `pointer-events: none`
- Preview — первое изображение работы (`images[0]`)

### Мобайл (< 768px)
Canvas заменяется вертикальным списком:
- Строки: `[IMG 60×48px] Название · год → ` 
- Разделитель `1px solid #1e1e1e` между строками
- Новые работы вверху (сортировка `year desc`)
- **Пагинация** (не бесконечный скролл): 5 работ на страницу
  - Кнопки `← Новее` и `Старее →` — pills с border white
  - Первая страница = самые новые
  - Активная кнопка отключается на крайних страницах (border #333, text #333)

## 12. SEO мета-разметка

### app/layout.tsx
```ts
export async function generateMetadata(): Promise<Metadata> {
  const settings = await getSettings()
  return {
    title: {
      default: settings.seo?.title ?? 'Влад Добровольский',
      template: '%s — Влад Добровольский',
    },
    description: settings.seo?.description,
    openGraph: {
      images: settings.seo?.ogImage ? [urlFor(settings.seo.ogImage).url()] : [],
      locale: 'ru_RU',
      type: 'website',
    },
    robots: { index: true, follow: true },
  }
}

// JSON-LD в <head>
const jsonLd = {
  '@context': 'https://schema.org',
  '@type': 'Person',
  name: 'Влад Добровольский',
  jobTitle: 'Звуковой художник, педагог',
  url: 'https://[домен]',
}
```

### works/[slug] и publications/[slug]
```ts
export async function generateMetadata({ params }): Promise<Metadata> {
  const item = await getItem(params.slug)
  return {
    title: item.seo?.title ?? item.title,
    description: item.seo?.description ?? item.description,
    openGraph: {
      title: item.seo?.title ?? item.title,
      images: item.seo?.ogImage
        ? [urlFor(item.seo.ogImage).url()]
        : item.images?.[0] ? [urlFor(item.images[0]).url()] : [],
    },
  }
}
```
